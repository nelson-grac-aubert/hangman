# Hangman
The popular word game made into a simple app to play on PC, Mac and Linux! 

## Requirements 

Python 3, using the PyGame library

## Features 
- Play a game of hangman with a word picked randomly from a list of french words
- Consult the list of possible words (we advise against it if you're a player, as it makes the game easier)
- Add or remove words from the list
